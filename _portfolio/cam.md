Gradient weighted class activation mappings use the gradients of any target concept to produce a localization map. An efficient implementation in PyTorch has been shown here.


The papers used for inspiration here are:
- [Learning Deep Features for Discriminative Localization](http://cnnlocalization.csail.mit.edu/Zhou_Learning_Deep_Features_CVPR_2016_paper.pdf)
- [Grad-CAM](https://arxiv.org/pdf/1610.02391.pdf)


The link to the Github Repository can be [found here.](https://github.com/deepandas11/Class-Activation-Mappings)
